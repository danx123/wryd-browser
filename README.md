WRYD Browser (Enterprise Edition)

WRYD (What Are You Doing) is an experimental, enterprise-oriented browser built as a derivative of Shrine Browser Lite, designed to explore a hybrid browser architecture using PySide6 (Qt for Python) and dual rendering engines.

WRYD is not a browser built from scratch. It is a curated, re-engineered evolution of Shrine Browser Lite, with selected features streamlined, redesigned, and optimized for a futuristic, cyberpunk-inspired experience.
